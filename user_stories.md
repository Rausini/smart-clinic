# User Stories — Smart Clinic Management System

## Patient

- **US-P1:** As a patient, I want to register an account so that I can access the clinic's services.
- **US-P2:** As a patient, I want to log in securely so that my personal data is protected.
- **US-P3:** As a patient, I want to view a list of available doctors so that I can choose the right specialist for my needs.
- **US-P4:** As a patient, I want to search doctors by name or speciality so that I can find the most suitable professional quickly.
- **US-P5:** As a patient, I want to book an appointment with a doctor so that I can schedule my medical visit.
- **US-P6:** As a patient, I want to cancel an appointment so that I can free the slot if I can no longer attend.
- **US-P7:** As a patient, I want to view my appointment history so that I can keep track of my visits.
- **US-P8:** As a patient, I want to view my prescriptions so that I can follow the treatment prescribed by my doctor.

---

## Doctor

- **US-D1:** As a doctor, I want to log in to the system so that I can manage my schedule and patients.
- **US-D2:** As a doctor, I want to view all appointments assigned to me so that I can plan my workday.
- **US-D3:** As a doctor, I want to mark an appointment as completed so that the system reflects the current status of each consultation.
- **US-D4:** As a doctor, I want to create a prescription for a patient after a consultation so that the patient has a formal record of their treatment.
- **US-D5:** As a doctor, I want to view all prescriptions I have issued so that I can track the treatments I have recommended.
- **US-D6:** As a doctor, I want to view a patient's profile so that I have the context needed before a consultation.

---

## Admin

- **US-A1:** As an admin, I want to log in with elevated privileges so that I can manage the entire system.
- **US-A2:** As an admin, I want to register new doctors so that the clinic's medical team is kept up to date.
- **US-A3:** As an admin, I want to remove a doctor from the system so that inactive or departed professionals no longer appear.
- **US-A4:** As an admin, I want to view all registered patients so that I have a complete overview of the clinic's user base.
- **US-A5:** As an admin, I want to view all appointments across the clinic so that I can monitor overall scheduling activity.
- **US-A6:** As an admin, I want to ensure that only authorized roles can access sensitive endpoints so that data security and access control are enforced.
